A summary of CSS

CSS is Cascading Style Sheets in full. It is the code that is used for styling web content. CSS is a markup language, just like HTML.

Anatomy of a CSS rulesheet

The selector defines the elements to be styled.

The declaration specifies which of the element properties you want to style.

Properties are ways in which you can style an HTML element.

The property value value chooses one out of many possible appearances for a given property.
